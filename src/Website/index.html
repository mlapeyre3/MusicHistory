The Project

Do music tendencies reflect historical events ? This is the question we would like to answer ! Our goal is to analyse different music features across time in order to detect sharp variations reflecting some major historical events and eventually visualize them. Visualisation of historical events thanks to the evolution of music features is an innovative approach. We would like to focus on the 20th century in the USA. Once this is achieved we could expand our visualisation to all EnglishÂ­speaking countries. Another improvement would be to add music samples to illustrate the visualization.

So let's see what's going on here !

The Dataset

To answer this problem we need a huge database with a lot of song and also many additional information for a given track.

The Million Song Dataset provides us the main information for one million song:

  • 1,000,000 tracks
  • 43,943 artists with at least one term
  • 54 fields per track
  • 2,201,916 asymmetric similarity relationships
  • 273 GB of data
  • 515,576 dated tracks starting from 1922
  • 44,745 unique artists
  • 18,196 cover songs identified
  • 7,643 tags
  • 2,321 unique musicbrainz tags

    • To fulfill the dataset, we also extract additional data from the musiXmatch dataset (237,662 lyrics) and additional tags for the classification were provided by the lastfm dataset

    Date is the key for an historical analysis. The most data we get, the better it is ! Therefore, we also used the spotify API which allow us to get additional date for 13% of the track
